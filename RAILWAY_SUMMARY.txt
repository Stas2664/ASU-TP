ШАГ 1: СОЗДАНИЕ БД НА RAILWAY
═════════════════════════════

1. Открой https://railway.app
2. Нажми "Start a New Project"
3. Войди через GitHub (или email)
4. Нажми "New Project"
5. Выбери "Provision PostgreSQL"
6. Дождись создания (~30 секунд)

✅ PostgreSQL база создана!


ШАГ 2: ПОЛУЧЕНИЕ ДАННЫХ ПОДКЛЮЧЕНИЯ
════════════════════════════════════

1. В Railway кликни на созданную PostgreSQL базу
2. Перейди на вкладку "Connect"
3. Скопируй данные:
   • Host
   • Port
   • Database
   • User
   • Password

✅ Данные скопированы!


ШАГ 3: АВТОМАТИЧЕСКОЕ РАЗВЕРТЫВАНИЕ
════════════════════════════════════

Запусти: DEPLOY_TO_RAILWAY.bat

Скрипт:
  • Спросит какой SQL файл загрузить
  • Попросит ввести данные подключения из шага 2
  • Автоматически загрузит схему БД
  • Проверит развертывание
  • Создаст файл .env.railway с настройками

✅ База данных развернута!


ШАГ 4: ПРОВЕРКА
═══════════════

Запусти: CHECK_RAILWAY_STATUS.bat

Скрипт покажет:
  • Список схем
  • Количество таблиц
  • Пользователей
  • Размер БД

✅ Все работает!

Если не хочешь устанавливать psql:

1. Railway → PostgreSQL → вкладка "Data"
2. Кнопка "Query"
3. Открой файл railway_init.sql в Notepad
4. Скопируй всё (Ctrl+A, Ctrl+C)
5. Вставь в Query Editor (Ctrl+V)
6. Нажми "Execute"
7. Готово!
